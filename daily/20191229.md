```
跳转：
1.alt + 数字:各个窗体跳转
2.ctrl+alt+左右键:各个代码跳转
3.ctrl+shift+A(help->find Actions):findActions eg:输入recent file最近打开文件 recently changed file 最近修改文件 
4.ctrl+E 最近打开文件
5.ctrl+shift+E 最近修改文件
6.ctrl+shift+backspace(Navigate->last Edit location):跳转到上次编辑的地方
7.ctrl+alt+左右键：跳转到上次浏览的位置。
8.F11(find action:toggle bookmark ):在代码上做标签，ctrl+F11 标记数字的方式做标签
  shift+F11显示标签
9.alt+shift+F(find action：add to favorites):收藏类或方法
10.emacsideas插件安装：(find action:plugins打开插件)：并不好用！
11.alt+1:跳转到文件区，esc跳转回编辑区
查找：
1.ctrl+N(navigate->class)：找到搜索的类类(jar包中的类：include non-project classes)
2.ctrl+shift+N(navigate->file)：找到搜索的文件
3.ctrl+shift+alt+N(navigate->symbol):找到搜索的方法符合
4.ctrl+shift+F(Edit->find->find in path):找到具体的字符串(match:是否匹配大小写，words是否搜索的字符串是个单词，regex正则表达式搜索，file mask在那类文件中搜索)

列操作：
1.ctrl+右键(find action:move carte to next word):光标移到下一个单词
2.home(find action:move caret to line start):光标移动到最前;end：最后
3.shift+home:光标选中当前所在位置到最前位置;shift+end:选中光标位置到最后
4.ctrl+shift+alt+J(edit->find->select all occurrences):选中所有相同字符串
  alt+j:选中下一个相同字符；alt+shift+j:取消
5.ctrl+shift+U(eidt->toggle case):单词大小写更换
6.ctrl+alt+L(code->reformat code)：自动对齐
7.shift+F6(Refactor->rename):变量重命名
8.ctrl+alt+V(Refactor->Extract->Variable):抽取静态变量
  ctrl+alt+C(Refactor->Extract->Variable):抽取静态变量为静态常量
  ctrl+alt+F(Refactor->Extract->Variable):抽取静态变量为成员变量
  ctrl+alt+P(Refactor->Extract->Variable):抽取方法参数
  ctrl+alt+M(Refactor->Extract->Variable):抽取函数
快速构建代码(有时间仔细研究一下)：
1.live templates(find action:live templates)设置live templates:
  psfi:public static final int $var1$ = $var2$;
  ...
2.postfix completion(find action:postfix completion)
  sout: System.out.println();
  100.fori：
  data.return:return data;
  data.nn:if (data != null) {};
  ...
3.alt+enter:(show intention Actions):自动创建函数;list relpace;字符串format或者build;实现接口;单词拼写;导包
```
